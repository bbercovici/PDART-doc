Detailed Description

Worker class. Provides interface between SbgatCore, SbgatGUI and QThreads.

Author
Benjamin Bercovici
Date
July 22, 2017

This class is used an interface to a QThread, enabling heavy data processing outside of the main thread within which the GUI lives. This allows to the GUI to remain responsive. Its implementation was based on the very informative blog post by Maya Posch at https://mayaposch.wordpress.com/2011/11/01/how-to-really-truly-use-qthreads-the-full-explanation/
