SBGAT

This GitHub Page website stores the documentation of the current version of SBGAT, the implementation of the Small Bodies Geophysical Analysis Tool.

SBGAT relies on QT and VTK to enable the user to interact with face/vertex shape models. The mindset behind SBGAT is to facilitate iterative modification and analysis of small body shape models by providing users with an intuitive interface allowing them to

  • Apply geometrical transforms to the shape models
  • Run science algorithms (such as gravity field generation, synthetic lightcurve generation,...)
  • Facilitate results visualization

SBGAT supports *.obj shape files restricted to vertex/facets information with standard vertex numbering (CCW indicates outward normal). There is no need to provide supplemental information such as surface normals since those will be recomputed internally, enforcing data consistency when surface alteration are introduced.

As of now, dependencies include

  • The Visualization Toolkit (VTK), an open-source, freely available software system for 3D computer graphics, image processing, and visualization. It consists of a C++ class library and several interpreted interface layers including Tcl/Tk, Java, and Python (http://www.vtk.org/)
  • Qt (/kjuÀêt/ "cute"), a cross-platform application framework (https://www.qt.io/)
  • Armadillo, a high quality linear algebra library (matrix maths) for the C++ language, aiming towards a good balance between speed and ease of use (http://arma.sourceforge.net/)
