Detailed Description

SurfacePGMWindow class defining a window where a user evaluate the Polyhedron Gravity Model of a shape model.

Author
Benjamin Bercovici
Date
October, 2018

Enables computation of surface potential, inertial accelerations, body-frame accelerations and surface slopes
