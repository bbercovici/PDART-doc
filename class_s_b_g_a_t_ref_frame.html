Detailed Description

Defines the SBGATRefFrame class.

Author
Benjamin Bercovici
Jay McMahon
Date
October 2018

Defines the SBGATRefFrame class, a right-handed set of basis vector positioned at some origin in space. The position of the origin and the orientation of the basis vectors are specified with respect to the parent frame of the present frame.
