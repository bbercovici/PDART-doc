Detailed Description

Mainwindow class. This is the main class of the SbgatGUI application.

Author
Benjamin Bercovici
Date
July 22, 2017

{Main class of the GUI as it hosts the VTK pipeline visualizer and the actions/menus allowing the user to interact with the program data. This class inherits from QMainWindow and hosts the QVTKOpenGLWidget where all rendering and display tasks occur. Also exposes SbgatCore classes to the user through the user interface layer brought by Qt.}
