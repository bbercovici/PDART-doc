Detailed Description

YORPWindow class defining a window where a user can specificy the inputs to a YORP coefficients computation.

Author
Benjamin Bercovici
Date
March, 2018

This class inherits from QDialog and enables the user to set a number of options used in the YORP coefficients calculation. The calculation can only take place if a spacecraft or a small body shape model is available and if an output directory has been specified
